# Sample vault

This vault is intentionally populated for local plugin development.

## Areas

- `daily/` keeps daily notes.
- `projects/` keeps active work.
- `reference/` keeps command examples and design notes.
- `archive/` keeps stale material.

## Search hints

- Search `TODO` to verify default matching.
- Search `todo` with `ignore-case` to compare results.
- Search within `projects/` by passing `path=projects/`.
- Search `grep` with `line-number` to verify formatting.
